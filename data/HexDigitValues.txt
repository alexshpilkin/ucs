# This file is a supplement to the Unicode Character Database.
#
# It lists numeric values for hexadecimal digits, that is, characters with
# Hex_Digit=Y, where those are not specified in the Unicode Character Database
# proper. This covers digits with values 10 to 15 decimal, which are the small
# and capital Latin letters A to F and their fullwidth equivalents.
#
# The values here are listed in the same way as in DerivedNumericValues.txt.
# However, they should not be used together except when parsing a hexadecimal
# numeral, because in Roman numerals some of the same Latin letters have
# different values.
#
# For more information, see the Unicode FAQ entry on hexadecimals,
# at https://unicode.org/faq/casemap_charprop.html#19
#
# This file is valid for Unicode versions 2.0.0 through 12.0.0.
